# Logistics-Analysis-Dashboard
### Excel + PowerBI Project | Orders Delivery Data | From Raw Data to Businesss Insights.

### Project Overview
Logistic Dashboard used to assess the efficiency of each component of a logistics company, comprising of a network of Hubs, a fleet of delivery vehicles, and a number of drivers. Each component is analyzed in detail in order to surface insights and recommendations to optimally run the organisation

### Tools & Technologies
* SQL
* Excel
* PowerBI

### Dataset Overview
Columns:
Date, Product_ID, Product_Name, Category, Units_Sold, Unit_Price, Total_Sales, Cost_Price, Profit
